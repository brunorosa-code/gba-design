# GBA Screen Templates

This folder holds reusable screen templates for the Global Bank Account (GBA) prototype.

Use these templates as a starting point when designing new flows for each country. Copy or adapt them into the country-specific folders (`gba-brasil/`, `gba-mexico/`, etc.) as needed.

## How to use

- Add new template files here (e.g. layout components, common screen structures).
- Keep templates generic so they can be reused across countries.
- Country-specific customizations live in `gba-{country}/screens/`.
